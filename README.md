### Run it locally

```bash
$ curl -LOk https://github.com/maxatbrs/unixtimestampme/archive/master.zip
$ unzip master.zip
$ mv unixtimestampme-master/ my-project;
$ cd my-project;
$ python -m SimpleHTTPServer 8000
```

Browse to http://localhost:8000 to see it in action!
